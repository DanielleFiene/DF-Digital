Services

I create fast, reliable and user-friendly digital products — from complete SaaS platforms to custom web applications — with a focus on performance, security, and real-world results.

SaaS & Web Applications

Complete platforms from scratch: user authentication, APIs, databases, payment integration (Stripe), cloud deployment. Multi-tenant architecture, real-time features, admin dashboards.
Stack: TypeScript, React, Next.js, Node.js, PostgreSQL, Prisma

Geospatial & Mapping

Advanced mapping features, geocoding systems, spatial analysis, custom search engines, vector tile optimization, real-time filtering, data visualization.
Stack: Mapbox GL JS, Turf.js, Leaflet, geocoding APIs

API Development & Integration

RESTful APIs, third-party integrations (WhatsApp, OpenAI, Stripe, Google Maps, etc.), webhooks, background jobs, concurrent data processing.

Cloud Infrastructure & DevOps

Google Cloud Platform (Cloud Run, Cloud SQL, Artifact Registry), Docker containerization, CI/CD pipelines (GitHub Actions), monitoring & alerting, SSL/TLS setup.

Performance Optimization

Performance auditing (Lighthouse/Core Web Vitals), speed optimization, code splitting, caching strategies, and accessibility improvements. Technical improvements for faster load times and better user experience.

Maintenance & Support

Ongoing feature development, bug fixes, performance optimization, security updates, production support. Available on a flexible or retainer basis.
